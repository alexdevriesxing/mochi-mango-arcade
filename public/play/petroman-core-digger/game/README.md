# Petroman: Core Digger Deluxe

A polished original HTML5 dig-and-chase arcade game for Mochi Mango Arcade.

## Production revision

- Expanded native 1280×720 presentation.
- Large 36×18-cell playfield with 34×16 playable interior.
- Twelve redesigned stages with broad loops, escape routes and readable boulder traps.
- Purifier orb now follows tunnels, ricochets and returns to Petroman.
- Enemies emerge from the reactor hatch in paced waves.
- Smog creatures progressively gain digging behaviour.
- Reactor drums can be pushed and dropped for escalating crush chains.
- Four stage-clear routes: collect all cores, defeat all enemies, spell PETRO, or collect the rare Golden Core.
- Input buffering and faster movement improve keyboard and touch responsiveness.
- All runtime character, enemy, object, terrain, VFX, background, HUD and screen art uses packaged hand-drawn raster assets.

Open `index.html` through a static web server.
