# Credits and Rights

Petroman, Petro City, Smog King, all illustrated sprites, objects, UI, environments, audio and game code in this package are original assets produced for the Petroman universe. The game uses no artwork, characters, layouts, music, code or branding from Mr. Do! or any other commercial arcade title. The design preserves the broad dig-and-chase arcade genre while using original stages, enemies, scoring, visual identity and progression.
