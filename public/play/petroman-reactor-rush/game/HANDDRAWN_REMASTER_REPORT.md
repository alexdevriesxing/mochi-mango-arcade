# Petroman: Reactor Rush — Hand-Drawn Remaster

This package replaces the earlier geometric/procedural-looking visual set with final illustrated assets.

## Runtime-integrated replacements
- 12-frame illustrated Petroman animation sheet
- 3 illustrated enemy families, four animation frames each
- 8 painted stage backgrounds
- 8 illustrated industrial platform tile strips
- Illustrated core, purifier, life, clock and bonus atlas
- Illustrated collection, impact and purifier VFX atlas
- Illustrated HUD icon atlas
- Remastered title, victory and defeat scenes
- Full-resolution source art retained under assets/source-art

All runtime paths remain local and dependency-free. No placeholder image references are used.
