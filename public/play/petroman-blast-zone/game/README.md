# Petroman: Blast Zone

A complete, original HTML5 bomber-maze arcade game for Mochi Mango Arcade.

## Launch

Upload the folder to any static web host and open `index.html`.

No build step, CDN, package manager, server-side runtime, or external asset host is required.

## Controls

- Move: Arrow keys or WASD
- Place Purifier Charge: Space, X, or K
- Remote detonation after finding the upgrade: C or L
- Pause: P or Escape
- Start / continue: Enter or tap the canvas
- Responsive touch controls are included for mobile devices

## Game structure

- 24 handcrafted zones
- 6 visually distinct worlds
- 6 boss encounters, including a two-boss finale
- Destructible smog blocks and hidden exits
- Chain-reaction Purifier Charges
- Bomb kicking and remote detonation upgrades
- Flame, charge-capacity, speed, kick, and remote upgrades
- Hidden reactor cores and PETRO letters
- Extra-life reward for completing PETRO
- Teleport pads, conveyors, ice, timed hazards, and fortress gates
- Persistent local high score
- Dedicated title, stage-clear, pause, victory, and defeat screens

## Art policy

All visible runtime game elements are packaged raster assets. Petroman, enemies, bosses, bombs, explosions, floor tiles, walls, destructible blocks, environment backgrounds, pickups, UI panels, HUD elements, and end screens are drawn from the approved hand-illustrated Petroman asset set and wired into the runtime.

Dynamic score values, timers, objective counts, and instructional labels are rendered as text over the illustrated UI.

## Originality

This is an original Petroman-universe game. It uses the broad maze-bombing arcade genre conventions of timed charges, destructible obstacles, chain reactions, power-ups, enemies, and exits. It does not include Bomberman characters, names, stage layouts, graphics, music, sound effects, or code.
