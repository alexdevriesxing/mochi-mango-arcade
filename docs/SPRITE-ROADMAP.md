# Sprite Rollout Roadmap

_Generated by `scripts/build-sprite-roadmap.mjs` — do not hand-edit; re-run after changing `games.json`._

## Position

- **146** spritesheets delivered (1920×1920 RGBA, background already at alpha 0).
- **74** map to an existing mascot by name; **72** are orphans awaiting a mascot remap.
- **159 / 392** games are covered by a name-matched character today.

Sheets are laid out in semantic rows (idle → walk → run → jump/dash → action → emote → face)
but row and column counts vary per sheet, so frames are segmented on the alpha channel
rather than sliced on a fixed grid. See `scripts/slice-spritesheet.mjs`.
